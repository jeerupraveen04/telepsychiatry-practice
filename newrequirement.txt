General Form Usability Updates
Please apply the following usability updates across the form where applicable.
1. Phone and Fax Fields
Phone and fax fields should not be required.
Many patients do not know these numbers.
Examples include (but are not limited to):
•Previous psychiatric provider phone number
•Previous provider fax number
•Therapist phone number
•Therapist fax number
•Pharmacy fax number
•Hospital or facility phone numbers
These fields should remain optional.
Patients should be able to proceed without entering this information.
2. Date Fields (Hospitalizations, Labs, Visits, etc.)
For date questions such as:
•Last psychiatric hospitalization
•Last medical hospitalization
•Most recent laboratory testing
•Last visit with a previous psychiatric provider
•Last therapy visit
•Last substance use treatment episode
Patients may not remember exact dates.
Please allow the following options.
Option A (preferred)Add a checkbox option:
☐ Date unknown
If selected, the date field should no longer be required.
Option B (acceptable alternative)
Allow the patient to type text into the date field such as:
•Unknown
•Approximate
•Not sure
3. Medication Dose Fields
Patients frequently do not remember exact medication doses.
For fields such as:
•Highest dose taken
•Current dose
•Previous dose
•Dose at time medication was stopped
Please update the field label to read:
Highest Dose (if known)
And allow patients to enter text such as:
•Unknown
•Not sure
•Approximate
Dose fields should not prevent completion of the form if the patient does not know
the dose.
Implementation Notes• These updates are intended to reduce form abandonment and improve completion
accuracy.
• Patients should be able to continue even if these details are not available.
• Required fields should remain limited to information necessary for clinical evaluation.

sections if they remember additional details they would like to include.
